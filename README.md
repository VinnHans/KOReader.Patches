# KOReader Patches
This is a collection of patches I've made for KOReader
## How to install
[Check out this link](https://koreader.rocks/user_guide/#L2-userpatches) to learn how to install patches.
## [2-trophy-bookmark](https://github.com/VinnHans/KOReader.Patches/blob/main/2-trophy-bookmark.lua)
<img width="300" height="400" alt="Rounded corners to book covers" src= "https://github.com/VinnHans/KOReader.Patches/blob/main/Assets/FileManager_2026-05-25_193948.png?raw=true" >
Adds a bookmark with a trophy to your finished books. You'll need to add the `trophy.bookmark.svg` in the `koreader/icons` folder.
Theres both a black and white and a color option avalible. This patch was put together by putting 2 different patches together.

Created with [2-percent-badge.lua by SeriousHornet ](https://github.com/SeriousHornet/KOReader.patches/blob/main/2-percent-badge.lua) and [2-disable-fullyread-progressbar.lua by joshucant](https://github.com/joshuacant/KOReader.patches/blob/main/2-disable-fullyread-progressbars.lua)
